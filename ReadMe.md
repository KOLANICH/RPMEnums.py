Just pythonic access to RPM enums, like

```python
import RPMEnums
print(RPMEnums.RPMBuild.CHECK | RPMEnums.RPMBuild.CLEAN) # <RPMBuild.CLEAN|CHECK: 24>
print(RPMEnums.RPMBuild(24)) # <RPMBuild.CLEAN|CHECK: 24>
```
